Project 3: Web Development Basics

For this project, I built the professional website you are currently viewing. The goal was to create a digital home for my work while learning the fundamentals of modern web development and version control.


What I did to finish it:

  • Setup: I configured Git and GitHub to track my progress and keep a history of my changes.

  • Jekyll: I used the Jekyll framework to make the site easier to manage and organize.
  • Hosting: I deployed the site using GitHub Pages, making it accessible to anyone on the web.
  • AI: I used AI tools to assist in debugging and optimizing my code.

This project helped me move from basic Scratch logic to building a real-world application used by developers today.
